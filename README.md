# Project Generator (V2) - Consultant Studio

## Overview
A consultant's product studio that transforms project ideas into deployable prototypes through structured intake, data modeling, and intelligent code generation. The platform enables AI implementation consultants to rapidly build client-ready prototypes for PE/VC engagements.

## Problem
Consultants spend days on project setup and scaffolding when they need to rapidly prototype client solutions and demonstrate technical feasibility during sales cycles

## Target User
AI implementation consultants who build tools for themselves and present prototypes to PE/VC clients as part of selling and delivering consulting engagements

## Features
- Multi-step structured intake interview with API integration mapping
- Automated Postgres data model generation with external ID columns
- Complete site map generation with auth requirements and API call mapping
- Enhanced scaffold generator producing migration files, typed queries, and integration stubs
- Three UI template options (Clean SaaS, Dashboard Heavy, Marketing Plus App)
- Project feedback notes system for collaboration
